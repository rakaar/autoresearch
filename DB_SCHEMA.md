# DB SCHEMA

## Purpose
This file defines the minimum information that every run must record.

## Required fields per run
- run_id
- date
- track_id
- parent_run_id if any
- short title
- idea summary
- what was run
- main result
- verdict
- artifact paths
- code/data reference
- notes for next step

## Required fields per track
- track_id
- title
- hypothesis or strategy
- status
- best_run_id if any
